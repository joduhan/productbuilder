개인정보처리방침

최종 업데이트: 2026년

본 사이트("타로 리딩")는 별도의 회원가입이나 로그인 없이 이용할 수 있는 오락용 웹 서비스입니다. 본 방침은 사이트 이용 중 발생할 수 있는 정보 처리 방식을 안내합니다.

1. 자체적으로 수집하는 정보

본 사이트는 별도의 서버나 데이터베이스를 운영하지 않는 정적 사이트입니다. 회원가입, 로그인, 결제 등 개인 식별 정보를 직접 수집하지 않습니다.

2. 브라우저에 저장되는 정보 (로컬 스토리지)

타로 카드를 뽑을 때마다 날짜, 리딩 종류, 뽑힌 카드 정보가 사용자의 브라우저(localStorage)에만 저장되어 "지난 기록" 기능에 사용됩니다. 이 정보는 외부 서버로 전송되지 않으며, 브라우저 저장공간을 삭제하면 함께 삭제됩니다.

3. 제휴 문의 양식 (Formspree)

제휴 문의 양식을 작성해 제출하면 입력하신 이름, 이메일, 문의 내용이 문의 접수를 위해 외부 서비스인 Formspree로 전송됩니다. 해당 정보의 처리에 대해서는 Formspree의 개인정보처리방침이 적용됩니다.

4. 댓글 기능 (giscus / GitHub Discussions)

댓글 기능은 giscus를 통해 GitHub Discussions로 운영됩니다. 댓글을 작성하려면 GitHub 계정으로 로그인해야 하며, 작성된 댓글은 GitHub에 공개적으로 저장됩니다. 댓글을 남기지 않으면 GitHub 계정 정보가 수집되지 않습니다.

5. 광고 (Google AdSense)

본 사이트는 Google AdSense를 통해 광고를 게재할 수 있습니다. Google 등 제3자 광고 제공업체는 쿠키를 사용해 사용자의 이전 방문 정보를 기반으로 맞춤 광고를 표시할 수 있습니다. 맞춤 광고를 원하지 않으시면 Google 광고 설정에서 개인 맞춤 광고를 선택 해제할 수 있으며, 자세한 내용은 Google 광고 정책에서 확인할 수 있습니다.

6. 문의

본 방침이나 사이트 운영과 관련해 문의하실 사항이 있으면 사이트 내 "제휴 문의" 양식을 통해 연락해 주세요.
